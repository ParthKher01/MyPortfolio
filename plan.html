Lifelong Learning Plan

I have an IT background, and I will be looking for skills that can help my personal growth and career. If I choose to learn one skill, then I go with coding skill. Coding is an essential skill nowadays. With the development of technology, everyone should know basic coding skills. I would like to learn coding because it aligns with my academic and professional goals. Coding enhances problem-solving, critical thinking, and other skills. It also creates many career opportunities.
Learning Goals: If I look at current trends, JavaScript is the first choice for web developers to start their web design or development journey.
Short Term Goals:
  • Currently, I have a beginner-level JavaScript knowledge, so I want expert level in JavaScript.
  • I enrolled in a full JavaScript course from popular platforms such as Udemy, Internshala, and Canvas, and W3School.
  • After that, I build a portfolio or simple project using HTML5, CSS3, and JavaScript.
  • I solve coding problems every day on platforms such as Lit Code.
  • It profoundly improves my coding skills. I know Git version control so I can upload my code on that platform to manage my projects effectively.
Long Term Goals:
  • I want to finish a few small projects in JavaScript language.
  • Gain good command over backend technologies such as NodeJS, Python, and PHP.
  • Once I get frontend and backend knowledge, I would finish some full stack projects to showcase in portfolio.
  • Work on small open-source projects that would be great work to have in portfolio.
  • Parallelly, I have experience in databases such as SQL, MySQL, and MongoDB to store and manage data.
  • Learn about API development and integration to enable data communication between applications.
  • Also, I want to learn WordPress and Wix platforms to create design websites.
Learning Activities
  • Enroll in Online Courses: It is easy to learn nowadays, internet makes learning very easy. I joined courses on popular platforms such as FreeCodeCamp, Udemy, InternShala, and W3School. These courses are well-designed and provide step-by-step guides that help in understanding concepts toughly.
  • Practice Coding: It is good to practice practically. Practice coding at least 1 hour regularly and also solve coding problems on Leet Code and CodeWars. Regular practice will improve programming logic and problem-solving skills.
  • Create Small Projects: Create small projects like simple mathematical operations, managing tasks lists, and personal blogs. Those projects will allow me to apply my learning in a practical part.
  • Read Books & Documentation: I will read JavaScript documentation and books like “Eloquent JavaScript” and “Advanced JavaScript” to understand programming better.
  • Contribute to Open-Source Projects: I will work on open-source projects to get real coding experience, learn from other developers, and improve my coding skills.
  • Build Soft Skills: I will practice communication, teamwork, and problem-solving to become a better developer. This includes explaining code clearly and working well with others.
Required Resources
  • Online learning platforms (FreeCodeCamp, Udemy, MDN Docs, YouTube tutorials)
  • Code editors (VS Code, Sublime Text)
  • Coding challenges websites (Leet Code, CodeWars)
  • Books ("Eloquent JavaScript," "Advanced JavaScript")
  • Community support (Reddit, GitHub)
  • A cloud-based repository (GitHub) to store and manage code projects.
  • Access to a test server for deploying projects and testing backend functionality.
  • A mentor or experienced developer to guide and provide feedback on my progress.
  • Hosting platforms such as hostinger, word-press, go-daddy so I can host my website online.
Monitoring Progress
  • Maintain a coding journal to track daily learning progress. There are many applications that can be helpful to keep my records.
  • Set weekly milestones to measure progress against learning goals.
  • Take quizzes and complete exercises from courses to assess understanding.
  • Track project completion and problem-solving efficiency.
  • Regularly review completed projects to identify areas of improvement.
  • Seek feedback from mentors and peers on coding style and best practices.
  • Maintain a GitHub repository with commits showcasing my learning journey.
  • Participate in online coding competitions to track my improvement over time.
Evaluation
  • Complete at least two coding courses with certificates.
  • Develop and deploy at least two projects (frontend and full stack).
  • Contribute code to an open-source repository.
  • Share learning progress through blog posts or a portfolio website.
  • Reflect on the journey by reviewing progress and setting future goals.
  • Assess personal improvement in problem-solving skills by comparing initial and current performance on coding platforms.
  • Conduct self-evaluations every three months to reassess learning goals and make necessary adjustments.
  • Present a coding project to peers or in an online community for review and feedback.
  • Apply for internships or freelance opportunities to gain real-world coding experience.
